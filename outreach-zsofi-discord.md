I wanted to share something I've been working on as part of my Aliit work. I drafted a MIP for a Midnight Agent Identity Standard (MAIS). It defines how AI agents can register, build reputation, and get validated on Midnight, all while respecting the three tier privacy model.

The short version is four Compact contracts. An Identity Registry with both public and private modes. A Reputation Registry where scores are public but evidence stays private. A Validation Registry with staked validators across three trust tiers. And a Disclosure Tier Registry that maps agent data to public, auditor, and private views. There is also an optional bridge to ERC-8004 for agents that want cross chain identity with Ethereum.

I posted an RFC on the forum and Discord and I am collecting feedback before submitting the formal PR to the MIPs repo.

Specifically I would love your thoughts on whether this fits what the Midnight Foundation is hoping to see from the Aliit program, whether there is any existing work on agent identity I should be aware of, and if there is a good way to get a community workshop going for a deeper review.
